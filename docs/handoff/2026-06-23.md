## What Was Tried and Abandoned
- **Approach**: Initially mounted both Recall editors (attempt + peek) without Monaco `path` props.
  - **Why we walked back**: Monaco shares a single text model across path-less editors, so the peek editor's `value={original}` populated the attempt editor and typing edited the original. Looked like "what I type affects the original."
  - **Lesson**: Multiple `@monaco-editor/react` `<Editor>` instances on screen at once MUST each get a unique `path`.

## Decisions Made
- Recall Mode owns the keyboard while open (modal); canvas shortcuts bail on `recallActiveRef`.
- Recall tests code only ‚Äî comments are stripped from the target and the peek.
- Persist recall progress per note in `data.recall` (wholesale node-data save, no schema/SQLite change).
